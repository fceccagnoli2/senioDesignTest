This is a directory that was used for recommendations research purposes.

-distribution_fitting.ipynb is an attempt to fit supplier performance (days late) to a distribution
-example_tabulated_month.csv is a file containing and example of how data is broken down and ingested by the recommendation algorithm. The data contained within is real.
-joint_dist.ipynb is a research file containing information on how suppliers within a material group are fit into a joint distribution.
-model_data files are all data files storing the information used in the research in this directory.
-MVEF.py is a script containing the functions used to generate supplier recommendations.
-old_mvef_work.ipynb was an early attempt at generating supplier recommendations.
-presentation_visuals.ipynb is a file that was used to generate images used in the final presentation.
-test.ipynb is a research file that shows an example of how suppliers are fit to distributions.
